# Helm charts Dev

This collection of Helm Charts is tailored for developers !
It is primarly designed to work with [Onyxia](https://github.com/inseefrlab/onyxia) but also works as a standalone Helm repository.  
See the repository [Helm charts datascience](https://github.com/inseefrlab/helm-charts-datascience) for a collection of charts tailored for datascientists.
See the other repository ([Helm charts](https://github.com/inseefrlab/helm-charts)) for infrastructure focused charts.  

To use the repo on helm (version 3+) :
```
helm repo add inseefrlab-dev https://inseefrlab.github.io/helm-charts-dev
```  

The repo is also browsable directly https://inseefrlab.github.io/helm-charts-dev/index.yaml

Contributions are welcome, feel free to open issues or submit pull requests :)
