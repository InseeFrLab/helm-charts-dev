<details>
  <summary># Python client</summary>

First define environment variables with your credentials:
```bash
export MILVUS_URI="https://{{ .Values.ingress.hostname }}:443"
export MILVUS_USERNAME="root"
export MILVUS_PASSWORD="{{ $password }}"
export MILVUS_TOKEN="root:{{ $password }}"
```

Then use them in your Python script to setup a MilvusClient:
```py
import os
from pymilvus import MilvusClient

client = MilvusClient(
    uri=os.getenv("MILVUS_URI"),
    token=os.getenv("MILVUS_TOKEN")
)

client.create_database(db_name="my_database")
```
